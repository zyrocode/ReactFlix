​
​
        You have a CRUD endpoint for comments on:
​
        https://strive-school-testing-apis.herokuapp.com/api/comments/
​
        This means you can GET, DELETE, POST, PUT data from there.
​
        The Comment structure is this:
​
        {
          "comment": "A good movie but definitely I don't like many parts of the plot",
          "rate": 3,
          "elementId": "tt1756545"
        }
​
        Where:
        - comment is the comment inserted by the user
        - rate is a value between 1 and 5
        - elementId is the imdbID of the movie / serie
​
​
  1. Create the main page from Netflix with all the styles / search bar etc
  2. Create, using components the several "galleries" for the movies (at least 3 galleries with 3 sagas you like (es.: Harry Potter, Lord of the Rings etc))
  3. Implement a working search bar that triggers the APIs and propose the result.
  4. Implement loaders, validators and error messages for the App
  5. [EXTRA] For the search, implement empty state and loaders
  6. [EXTRA] Add, when clicking on a movie, the Comment list for it
  7. [EXTRA] From the comment list, let the user add a comment to every movie
    8. ​
  8. [EXTRA] Sort every AJAX response per Year (from newer to older)
